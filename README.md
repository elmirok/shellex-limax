# Limax

Limax is the Shellex OS package manager app and command.

Current MVP: local `.sapp` install, installed package listing, package info, package removal, source repository records and repo update checks.

Run `limax help` in Shell for usage.

## Shellex package

- App id: `system.limax.local`
- Version: `1.0.0`
- Runtime: `host`
- Type: `system-app`
